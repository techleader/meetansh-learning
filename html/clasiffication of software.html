Classification Of Software

What is Software?

Software is a set of programs that is created to do a specific work. Softwares can be classified into 2 major categories :-
1. System Software
2. Application Software.

System System

System System is a type of computer program that is designed to run a computer's hardware and application programs. There are 3 types of system software :-
1. Operating system
2. Utility Software
3. Language translator

Operating Software

An Operating System (OS) is the most important and basic set of software required for a computer to function. It manages the hardware resources and provides some common services for the users.

Utility Programs

Utility programs are the software programs that are used for day to day maintenance and efficient running of a computer such as hardware, operating system, applicationsoftware and data storage devices.
